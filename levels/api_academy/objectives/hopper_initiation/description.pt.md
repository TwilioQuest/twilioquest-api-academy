# APIs remotas

O desafio para você é desbloquear quatro **baús de conhecimento**, um para cada uma das casas de estudante na Academia API. Este baú é patrocinado pela **Casa Hopper**, em homenagem à inovadora da ciência da computação [Grace Hopper](https://en.wikipedia.org/wiki/Grace_Hopper). Para desativar o mecanismo de bloqueio, você deve responder a algumas perguntas relacionadas a APIs remotas. Responda às perguntas à direita e clique em *HACK*.
